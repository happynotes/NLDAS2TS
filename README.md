#NLDAS2TS  - Convert NLDAS GRB data to Time-Series Table

Work with NLDAS, GNLDAS, NLDAS-2, FORCA, FORCB and so on.

REQUIREMENT: wgrib and/or wgrib2 is prerequiest for this program.
